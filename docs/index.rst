Welcome!
--------

Inspired by `FormatJS`_ and `ECMAScript 2023 Internationalization API (ECMA-402)`_,
FormatPHP provides an API to format dates, numbers, and strings, including
pluralization and translation. FormatPHP is powered by PHP's `intl extension`_
and integrates with `Unicode CLDR`_ and `ICU Message syntax`_ standards. It
requires `libicu`_ version 69.1 or higher.
